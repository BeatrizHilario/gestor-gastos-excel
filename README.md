# Gestor de Gastos Automático (Excel)

## Descrição
Aplicação que automatiza a organização financeira. O usuário insere os gastos mensais via console/interface e o sistema gera um arquivo `.xlsx` perfeitamente formatado.

## Tecnologias
* **Java**
* **Maven**
* **HTML**
* **CSS**
* **Spring Boot**

## Por que este projeto?
Desenvolvi esta ferramenta para aplicar lógica de organização de dados e manipulação de arquivos externos, resolvendo um problema real de gestão financeira pessoal e automatização de planilhas mantendo assim um padrão e a organização.
